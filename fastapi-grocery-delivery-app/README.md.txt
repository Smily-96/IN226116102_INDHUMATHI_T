# Grocery Delivery App (FastAPI)

This is a FastAPI backend project developed as part of internship training.

## Features
- REST API using FastAPI
- Pydantic validation
- CRUD operations
- Cart and Order system
- Search, Sorting, Pagination
- Swagger API testing

## How to Run

uvicorn grocery_main:app --reload

Open:
http://127.0.0.1:8000/docs
