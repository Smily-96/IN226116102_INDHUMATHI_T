# Fine-Tuning BERT on a Kaggle Dataset

## Overview
This project fine-tunes a BERT model for text classification using a Kaggle dataset.

## Concepts Covered
- Text preprocessing
- Train, validation, and test split
- BERT tokenization
- Fine-tuning BERT
- Accuracy, precision, recall, F1 score
- Confusion matrix
- Experiment comparison

## Experiments
1. Freeze all BERT layers and train classifier
2. Fine-tune last 2 BERT layers

## Tools Used
- Python
- Hugging Face Transformers
- PyTorch
- Google Colab
