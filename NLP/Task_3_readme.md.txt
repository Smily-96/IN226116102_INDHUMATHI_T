# Chatbot using Transformers

## Overview
This project builds a chatbot using a pre-trained transformer model (DialoGPT) from Hugging Face.

## Features
- Interactive chatbot
- Uses transformer-based model
- Generates dynamic responses
- Maintains conversation flow
- Exit condition included

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells
4. Interact with chatbot
