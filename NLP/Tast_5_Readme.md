# Fine-Tuning DistilBERT for POS Tagging and Chunking

## Overview
This project fine-tunes DistilBERT for two token classification tasks:
- Part-of-Speech (POS) Tagging
- Chunking

## Dataset
CoNLL-2003 dataset

## Concepts Covered
- Token classification
- Transformer fine-tuning
- Label alignment
- POS tagging
- Chunking
- Evaluation using seqeval

## Tools Used
- Python
- Hugging Face Transformers
- Datasets
- Seqeval
- Google Colab
