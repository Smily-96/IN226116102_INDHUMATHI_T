# AI Resume Screening System

## Overview
This project implements an AI-powered Resume Screening System using LangChain.

## Pipeline
Resume → Extract → Match → Score → Explain

## Features
- Skill extraction
- Tool extraction
- Experience analysis
- Matching logic
- Scoring (0–100)
- Explainable output
- Debugging improvements
